What this website does

Smart Auth Scraper analyzes a URL and tells you whether it contains a real authentication form.

Core behavior

  • Fetches the page with requests first, then falls back to Playwright when the page is JavaScript-heavy.
  • Detects login/auth forms, even when there is no traditional <form> tag.
  • Returns extracted HTML, detected fields, auth confidence, and a screenshot preview.

Features

  • Multi-provider AI support: OpenAI, Ollama Cloud, and Google Gemini.
  • Confidence score so you can quickly judge whether it is really a login page.
  • Formatted HTML snippet with one-click copy.
  • Screenshot capture so you can visually verify the analyzed form.
  • Popup warnings for search-result URLs, provider issues, and backend/network errors.

How to use it

  1. Paste a direct website login URL.
  2. Select an AI provider from the cards.
  3. Click Scrape to view the analysis, confidence, fields, and screenshot.
